
Kairos Schools - Next.js (app router) full starter

Included:
- app/ (Next.js app router) with pages: home, dashboard, competency, pedagogy, videos, login, contact
- components: Header (animated), Footer, KPI
- public/logo.svg (dark neon style)
- pages/api endpoints: /api/login, /api/reports (mock)
- styles/globals.css (dark theme)

Deploy:
1) Upload/replace these files in your GitHub repo
2) Vercel will detect Next.js and deploy automatically
3) Run locally: npm install && npm run dev

Note: This is a starter; wire real backend and auth for production.
