
Kairos Schools - Final Next.js (app router) starter (Dark theme)

Included:
- app/ (Next.js app router) with pages: home, dashboard, competency, pedagogy, videos, login, contact
- components: Header (animated), Footer, KPI
- public/logo.svg (dark neon style)
- pages/api endpoints: /api/login, /api/reports (mock)
- styles/globals.css (dark theme)

Deploy:
1) Upload/replace these files in your GitHub repo root
2) Vercel will detect Next.js (appDir) and deploy automatically
3) Locally: npm install && npm run dev

Note: Replace mock API with real backend before production.
