
Kairos Schools - Full Next.js Starter (Dark theme)
Includes:
- Animated Header (framer-motion)
- Login page (mock API)
- Teacher Dashboard, Students, Reports
- Pages: competency, pedagogy, videos, books, contact
- API endpoints: /api/login, /api/reports (mock)

How to deploy:
1) Upload contents of this ZIP to your GitHub repo (replace old files)
2) Vercel will detect Next.js and auto-deploy
3) Locally: npm install && npm run dev

Notes:
- This is a starter scaffold. Replace mock APIs with real backend before production.
